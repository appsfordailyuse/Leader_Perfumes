# LEADER Perfume Website

## GitHub Pages Par Upload Kaise Karein

### Step 1 — GitHub Account
- github.com par jaen aur login karein

### Step 2 — New Repository Banayein
- Green "New" button dabayein
- Repository name likhen: `leader-perfume` (ya jo bhi chahein)
- "Public" select karein
- "Create repository" dabayein

### Step 3 — Files Upload Karein
- "uploading an existing file" link dabayein
- **Pehle `index.html` drag karein**
- Phir **`images` folder ke andar ki sab files** upload karein:
  - black-1.jpg
  - black-2.png
  - black-3.png
  - pink-1.png
  - pink-2.png
  - pink-3.png
  - pink-4.png
- **IMPORTANT:** Images ko `images/` folder mein rakhein

### Step 4 — GitHub Pages Enable Karein
- Repository Settings → Pages
- Source: "Deploy from a branch"
- Branch: `main` → `/ (root)`
- Save karein

### Step 5 — Website Live!
- 2-3 minute mein aapki website live ho jaegi:
- URL: `https://[aapka-username].github.io/leader-perfume`
